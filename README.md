Project Description

Nice Gadgets is a modern, responsive online store for smartphones, tablets, and accessories.
It allows users to browse product categories, view detailed specifications, and explore new and popular devices through an intuitive and engaging interface.
The project demonstrates practical use of React, TypeScript, and React Context for state management, focusing on clean architecture and maintainable code.
